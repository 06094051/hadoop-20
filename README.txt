This is a version of code that runs on Facebook's realtime
distributed database clusters powered by Apache Hadoop. This software supports 
appends to HDFS files and concurrent reads and writes from HDFS files. 
The primary applications that use these clusters are Scribe and Apache HBase.

This code is based on Apache Hadoop 0.20-append.
